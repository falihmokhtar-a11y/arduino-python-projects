# Arduino + Python Projects

Collection of embedded systems projects combining Arduino and Python.

## Projects

### 1. Morse Code System
- Encode and decode Morse code (Python)
- Send Morse via serial communication
- Display Morse using LED (Arduino)

### 2. Voltage Measurement System
- Measure analog voltage using Arduino
- Calculate average and RMS values
- Visualize signal with oscilloscope (Python)

### 3. Light Sensor
- Read light intensity using phototransistor
- Send data from Arduino to Python
- Detect light (hell) or dark (dunkel)

## Technologies
- Python
- Arduino (C/C++)
- Serial communication
